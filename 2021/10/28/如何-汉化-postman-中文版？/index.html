如何 汉化 Postman 中文版？

发表于2021-10-28|更新于2022-06-24|Postman
|阅读量:

介绍

好多小伙伴，尤其是刚入门软件测试的，对英文版本的 Postman 很抵触，希望有个中文版的，不过很遗憾的是 Postman 只有英文版本，但凡事都有例外，某天在逛 github 的时候，无意间发现了一个 star 2.4k 的开源项目 Postman-cn,今天我们就来看看这个开源项目如何将英文版本的 Postman 汉化成中文版。

github 地址

https://github.com/hlmd/Postman-cn

下载并安装 Postman

windows

https://dl.pstmn.io/download/latest/win64
https://dl.pstmn.io/download/latest/win32

Mac

https://dl.pstmn.io/download/latest/osx

Linux

https://dl.pstmn.io/download/latest/linux64

官网下载

https://www.postman.com/downloads/

安装 Postman,下载好后直接操作安装即可

汉化

windows

下载Postman对应版本的汉化包app.zip文件，下载地址：https://github.com/hlmd/Postman-cn/releases
解压 app.zip，将解压后的app文件夹复制到C:\Users\D1M\AppData\Local\Postman\app-8.11.1\resources目录下，并重启Postman

Mac

下载Postman对应版本的汉化包app.zip文件，下载地址：https://github.com/hlmd/Postman-cn/releases
解压app.zip，将解压后的app文件夹复制到/Applications/Postman.app/Contents/Resources目录下，并重启Postman

你以为已经汉化成功了吗，No,还缺了最重要的一步

设置禁止更新,打死不要更新

Windows

直接删除根目录下的Update.exe文件即可
修改 hosts 文件，在 hosts 文件中加入0.0.0.0 dl.pstmn.io

Mac

点击设置，将更新下的自动下载主要更新设置为禁用

修改 hosts 文件，终端执行 vim /etc/hosts,添加0.0.0.0 dl.pstmn.io保存并退出
