Nessus 最新版破解教程_Nessus8.14安装教程_文未送插件

发表于2021-05-30|更新于2022-06-24|运维安全
|阅读量:

Nessus 简介

Nessus 是目前全世界最多人使用的系统漏洞扫描与分析软件。总共有超过75,000个机构使用 Nessus 作为扫描该机构电脑系统的软件。Nessus号称是世界上最流行的漏洞扫描程序,通常包括成千上万的最新的漏洞，各种各样的扫描选项，及易于使用的图形界面和有效的报告。

Nessus 下载

首先进NESSUS官网(https://www.tenable.com/downloads/nessus)，下载8.14版本的安装程序，你是WIN系统也好，IOSX系统也好，LINUX系统也好，爱装那个版本装那个版本。
进入官网下载地址，根据自己的操作操作系统选择下载的版本。本文以Centos7系列安装为例。

Nessus 安装

下载好安装包Nessus-8.14.0-es7.x86_64.rpm后，把下载好的安装包及所需插件等上传到要安装的机器上，然后安装

[root@ODSP-DB data]# rpm -ivh Nessus-8.14.0-es7.x86_64.rpm
警告：Nessus-8.14.0-es7.x86_64.rpm: 头V4 RSA/SHA1 Signature, 密钥 ID 1c0c4a5d: NOKEY
准备中… ################################# [100%]
正在升级/安装…
1:Nessus-8.14.0-es7 ################################# [100%]
Unpacking Nessus Core Components…

  • You can start Nessus by typing /bin/systemctl start nessusd.service
  • Then go to https://ODSP-DB:8834/ to configure your scanner

启动

[root@ODSP-DB data]# systemctl status nessusd

Nessus 初始化

非常重要的一步，安装完毕后，系统会要求打开浏览器初始化扫描器，请一定要选择nessus manager by sc（每个小版本可能不一样，但是核心就是选择管理扫描器的为安全中心SC），然后就要求你创建用户名和密码，根据自己爱好随意.

安装完成后进行系统设置，打开访问网址 https://127.0.0.1:8834，初始化扫描器。

根据提示步骤，选择 Managed Scanner

再选择 Tenable.sc

最后，点击继续，创建账号密码即可登录。

Nessus 破解

然后更新all-2.0.tar.gz 插件 然后到安装目录执行nessuscli update all-2.0.tar.gz更新插件和授权

[root@ODSP-DB data]# cd /opt/nessus/sbin/
[root@ODSP-DB sbin]# ./nessuscli update /data/all-2.0.tar.gz

[info] Copying templates version 202105110858 to /opt/nessus/var/nessus/templates/tmp
[info] Finished copying templates.
[info] Moved new templates with version 202105110858 from plugins dir.
* Update successful. The changes will be automatically processed by Nessus.

修改安装目录下的配置文件 plugin_feed_info.inc 然后在替换一共需要替换两处。

[root@ODSP-DB nessus]# vim plugin_feed_info.inc
PLUGIN_SET = “202105110858”;
PLUGIN_FEED = “ProfessionalFeed (Direct)”;
PLUGIN_FEED_TRANSPORT = “Tenable Network Security Lightning”;
[root@ODSP-DB nessus]# cp plugin_feed_info.inc /opt/nessus/lib/nessus/plugins/
[root@ODSP-DB nessus]# cp plugin_feed_info.inc /opt/nessus/var/nessus/

重新启动NESSUS服务，重新登陆控制台即可看到破解成功

systemctl restart nessusd
