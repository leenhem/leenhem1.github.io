ElasticSearch ES 服务部署 单机部署

发表于2021-05-13|更新于2022-06-24|运维ElasticSearch
|阅读量:

下载

官网下载地址：https://www.elastic.co/cn/downloads/elasticsearch
ElasticSearch历史版本下载：https://www.elastic.co/cn/downloads/past-releases

wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-7.10.1-linux-x86\_64.tar.gz

源文地址：

安装

[root@ODSP-DB data]# tar -xf elasticsearch-7.10.1-linux-x86_64.tar.gz
[root@ODSP-DB data]# ll
总用量 311644
drwxr-xr-x 9 root root 4096 12月 5 09:04 elasticsearch-7.10.1
-rw-r–r– 1 root root 318801277 5月 12 11:18 elasticsearch-7.10.1-linux-x86_64.tar.gz

[root@ODSP-DB data]# cd elasticsearch-7.10.1
[root@ODSP-DB elasticsearch-7.10.1]# ls
bin config jdk lib LICENSE.txt logs modules NOTICE.txt plugins README.asciidoc
[root@ODSP-DB elasticsearch-7.10.1]#

配置

修改ElasticSearch配置

cat config/elasticsearch.yml

cluster.name: docshare-cluster
network.host: 192.168.80.209
http.port: 9200
discovery.seed_hosts: [“192.168.80.209:9200”]
cluster.initial_master_nodes: [“192.168.80.209”]

修改系统参数

修改limits.conf

编辑 /etc/security/limits.conf，追加以下内容；

* soft nofile 65536
* hard nofile 65536

此文件修改后需要重新登录用户，才会生效

修改sysctl.conf

在/etc/sysctl.conf文件最后添加一行
vm.max_map_count=262144
执行命令
sysctl -p 使配置生效

添加用户

添加用户，以普通用户启动ElasticSearch
useradd es

启动

使用 bin/elasticsearch -d 启动ElasticSearch

[root@ODSP-DB elasticsearch-7.10.1]# su - es
上一次登录：三 5月 12 10:11:31 CST 2021pts/1 上
[es@ODSP-DB ~]$ cd /data/elasticsearch-7.10.1/
[es@ODSP-DB elasticsearch-7.10.1]$ bin/elasticsearch -d
