服务器如何运行 VSCode开发工具

发表于2021-06-06|更新于2022-06-24|开发
|阅读量:

程序员会通过云服务来加速软件开发效率，今天要介绍的这款开源工具让你IDE更加接地气，它可以在远程服务器上运行 VS Code ，使其通过浏览器访问，它就是——code-server

code-server本周在GitHub上十分受欢迎,（GitHub地址：https://github.com/cdr/code-server）

总而言之，code-server它可以让你：
在 Chromebook、平板电脑和笔记本电脑上使用一致的开发环境。
利用大型云服务器来加速测试、编辑与下载等。
节能减耗，保持电池寿命
所有密集型计算都在服务器上运行。
无需再运行多余的 Chrome 实例。

安装 code-server-3.10.2-amd64.rpm

去Github上下载吧

rpm -ivh code-server-3.10.2-amd64.rpm
Preparing… ################################# [100%]
package code-server-0:3.10.2-1.x86_64 is already installed

先运行一下

先执行 code-server 运行起来，发现监听的地址是127.0.0.1

code-server

修改配置文件

在配置文件~/.config/code-server/config.yaml 修改密码

cat ~/.config/code-server/config.yaml
bind-addr: 0.0.0.0:8088
auth: password
password: 4dec31d18a18da2a22cae1ac
cert: false

启动

[root@host-192-168-65-94 ~]# code-server
